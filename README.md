# Finanzverwaltung

Ein modulares Programm zur Verwaltung Ihrer Finanzen.

## Module

- **Dashboard**: Übersicht über alle Vermögenswerte und Ausgaben.
- **Bankkonten**: Pflege von Konten und manuellen Umsätzen mit Kategorien.
- **Kategorien**: Hierarchische Kategorien für Umsätze.
- **Daueraufträge**: Wiederkehrende Zahlungen, manuell auslösbar.
- **Versicherungen**: Verträge mit Werten und Details.
- **Wertpapiere**: Pflege von Wertpapieren mit Kursen und Historie.
- **Depots**: Transaktionen (Kauf/Verkauf) mit Kosten und Erträgen, Depotübersicht mit GuV.
- **Abonnements**: Verwaltung mit Kündigungsfristen und Kosten.
- **Immobilien**: Anschaffungs- und Zeitwerte.
- **Firmenbeteiligungen**: Pflege von Beteiligungen.

## Verwendung

Öffnen Sie `index.html` in einem Webbrowser. Daten werden im localStorage gespeichert.

## Struktur

- `index.html`: Hauptseite
- `styles.css`: Styles
- `app.js`: Hauptlogik
- `modules/`: Einzelne Module
